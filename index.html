Portfolio

This is a collection of the most important projects that I've developed during these last years. Most of my learning has come from developing and learning about programming by myself, and I am very commited with the work that I do.

Project Romboid

Project Romboid is the first commercial game that I've developed. It is going to be published on Steam in the near future. Project Romboid is a hard roguelite in which players have to survive waves of enemies while picking items that help them survive the waves.

Blups

Blups is an arcade mobile game in which every 10 seconds a new minigame appears. The development of the game is currently paused and there is only a demo available (with 5 minigames), but I have plans to publish it on the Play Store in the future.

Satellite Engine

Satellite Engine is a game engine made from scratch with C++ and OpenGl as a learning experience. I wanted to deep dive into the topic of game engine, and I thought the best way to do it was to build one by myself.

Shootingat Resolved

Shootingat Resolved is a short 2D rogue-like game in which your objective is to defeat the terrible boss that awaits for you at the end of the dungeon. It has been made with Unity2D and C#.

Rocky Eruption

Rocky Eruption is the game that I co-developed with a friend of mine for the Ludum Dare 49 Game Jam. It is a game in which players have to survive the lava by climbing the interior of a volcano. It was made with the Unity Engine.
